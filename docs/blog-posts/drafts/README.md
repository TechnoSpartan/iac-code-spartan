# Borradores de Blog Posts

Carpeta para posts en desarrollo. Cuando un post esté completo, moverlo a `published/`.

## Workflow

1. Copiar `TEMPLATE.md` como base
2. Desarrollar el contenido
3. Revisar y mejorar
4. Mover a `published/` cuando esté listo
